# YummyCart 🍔🛒

YummyCart is a fullstack e-commerce food ordering application.  
It includes a **React frontend** and an **Express + MongoDB backend**.

---

## **Features**

- User Registration & Login ✅  
- Browse Food Items 🍕🍔  
- Add to Cart & Remove Items 🛒  
- View Orders 📦  
- Responsive UI using Tailwind CSS
